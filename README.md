# DevConnect 🚀

DevConnect is a full-stack social networking platform for developers built with Next.js 13, TypeScript, Prisma, PostgreSQL, and NextAuth.

## Features

- Authentication (JWT Strategy)
- Public Developer Profiles
- Create / Delete Posts
- Like System
- Follow / Unfollow Users
- Search Developers
- Secure Authorization Checks
- Server & Client Component Architecture

## Tech Stack

- Next.js (App Router)
- TypeScript
- Prisma ORM
- PostgreSQL
- NextAuth
- Vercel Deployment
